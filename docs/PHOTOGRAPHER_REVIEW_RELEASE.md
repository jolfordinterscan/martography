# Martography v1.2 — Photographer Review Release

## Release intent

This release preserves the established Martography visual language while strengthening the site as
a discoverable wildlife archive. The recommendations below are editorial only. They do not change
featured flags, publication status, print availability, artist titles, or story content.

## Archive review

- Mammals, Birds, Insects, Arachnids, and Reptiles contain public photographs and appear as public
  category filters.
- Amphibians and Aquatic Life contain no imported photographs. They remain valid taxonomy values but
  are not exposed as empty public destinations.
- Every image-backed public wildlife record appears in the Gallery, its wildlife category, and its
  generated Species page.
- Species pages now use a leading photograph and reveal related collections and stories only when
  those relationships exist.
